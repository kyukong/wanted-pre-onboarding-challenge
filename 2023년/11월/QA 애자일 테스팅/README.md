# 원티드 프리온보딩 11월 QA 애자일 테스팅

## 강의 학습 이유

- 프로젝트 완성도를 위해 QA 를 진행하였으나 명확한 체계없이 진행
- 팀 단위 프로젝트를 진행할 때 애자일 방식에 대해 미숙하여 오히려 개발 지연 발생
- 체계있는 애자일 방법을 도입하고자 학습 진행
